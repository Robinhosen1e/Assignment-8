UrbanThread

A modern full-stack e-commerce web application built with Next.js for men's fashion — clothing, leather jackets, shoes, bags, and boots.

🔗 **Live Demo:** [https://assignment8-blush.vercel.app/](https://assignment8-blush.vercel.app/)  

---

## 📖 About the Project

UrbanThread is a responsive and feature-rich e-commerce platform specializing in men's clothing and accessories. The application allows users to browse products such as shirts, pants, leather jackets, shoes, sneakers, bags, and boots. It includes category filtering, search functionality, product details, and secure user authentication.

---

## ✨ Key Features

### 🏠 Home Page
- Hero Banner with swiper
- Popular Products
- Trending Products
- Discounted & Flash Sale sections
- Customer Reviews
- Footer

### 🛍️ Products
- Category-based filtering (Shirts, Pants, Leather, Shoes, Bags, Boots ইত্যাদি)

### 🔍 Product Details Page
- High-resolution product images

### 🔐 Authentication
- Email & Password authentication

- Profile management

### 👤 User Profile
- Display user information
- Update name and profile picture
- (Order history - upcoming)

### 🎨 UI & UX
- Smooth animations with Framer Motion
- Fully responsive design
- Toast notifications
